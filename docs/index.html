node-service-bus

Experiment on developing a message bus in nodejs

Idea is to build a Event-Driven Application Framework having as back bones an Event Bus based on RxJS and a simple and flexible Modules System to develop complex and scalable backend services.

Build project
  1. Ensure to have nodejs version >=12.20.x
  2. install base dependencies
    npm install
  3. prepare lerna project
    npx lerna clean
npx lerna bootstap
  4. build packages
    npx lerna run build
  5. start sample
    npx lerna run start --scope=rxbus-sample --stream

  • bus-core/lib
  • rxbus/src
  • rxmq/ts/channel
